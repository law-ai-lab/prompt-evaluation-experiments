# Prompt Evaluation Experiments

This repository explores how different prompt designs affect the quality of responses generated by large language models (LLMs).

The goal is to analyze how prompts influence:

* response accuracy
* reasoning quality
* clarity of explanation
* completeness of information

## Example Prompt Comparison

### Prompt A

Explain climate change.

### Prompt B

Explain climate change in three simple sentences for a high school student.

### Evaluation

Prompt B produces more structured and accessible responses because it specifies:

* target audience
* response length
* clarity expectations

This demonstrates how prompt structure can significantly improve response quality.
